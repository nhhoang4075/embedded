# Embedded Systems - STM32F4

Học lập trình hệ thống nhúng với vi điều khiển STM32F4.

## Yêu cầu phần mềm

| Phần mềm | Phiên bản | Mô tả |
|-----------|-----------|-------|
| STM32CubeIDE | 2.1.0 | IDE + compiler + debugger |
| STM32CubeMX | latest | Cấu hình ngoại vi và sinh code tự động |
| CubeF4 | latest | Thư viện mã nguồn cho STM32F4 (cài qua CubeMX) |

## Cài đặt

1. Tải và cài **STM32CubeIDE 2.1.0** từ [st.com](https://www.st.com/en/development-tools/stm32cubeide.html)
2. Tải và cài **STM32CubeMX** từ [st.com](https://www.st.com/en/development-tools/stm32cubemx.html)
3. Mở CubeMX → Help → Manage embedded software packages → cài **STM32CubeF4**

embedded/
